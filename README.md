# 🌍 Local Turístico - Busan

Projeto desenvolvido como desafio prático da **Rocketseat**, com foco em **HTML** e **CSS**, recriando uma landing page de turismo a partir de um layout no **Figma**.

## 📖 Sobre o projeto

A proposta deste desafio foi desenvolver uma página sobre um local turístico, destacando a cidade de **Busan**, na Coreia do Sul, com:

- título principal
- imagem de destaque
- texto introdutório
- 3 pontos turísticos
- rodapé com créditos

O objetivo foi praticar:

- estruturação semântica com HTML
- estilização com CSS
- tipografia
- espaçamentos
- bordas arredondadas
- organização visual baseada em layout do Figma

## 🚀 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **Google Fonts**

## 🎨 Layout

O projeto foi construído com base em um layout disponibilizado pela **Rocketseat** no **Figma**.
